JobSwipe 利用規約

最終改定日：令和8年7月1日

本利用規約（以下「本規約」といいます。）は、株式会社Level Frontier（以下「当社」といいます。）が提供する求人検索・求人マッチングサービス「JobSwipe」（以下「本サービス」といいます。）の利用条件ならびに当社と利用者との間の権利義務関係を定めるものです。

本サービスを利用するすべての利用者は、本規約の全文を確認し、本規約に同意したうえで本サービスを利用するものとします。

第1条（適用）

  1. 本規約は、本サービスの提供条件および本サービスの利用に関する当社と利用者との一切の関係に適用されます。
  2. 当社が本サービス上または当社ウェブサイト上で掲載するガイドライン、ヘルプ、注意事項、個別規定その他のルールは、本規約の一部を構成するものとします。
  3. 本規約の内容と前項のルールその他本サービスに関する説明等が異なる場合は、本規約の定めが優先して適用されます。ただし、個別規定において本規約に優先する旨が明示されている場合は、当該個別規定が優先します。

第2条（定義）

本規約において使用する用語の定義は、次の各号のとおりとします。

  1. 「本サービス」とは、当社が提供する「JobSwipe」という名称の求人検索・求人マッチングサービスをいいます。理由の如何を問わずサービス名称または内容が変更された場合は、変更後のサービスを含みます。
  2. 「利用者」とは、本サービスを利用するすべての個人、法人または団体をいいます。
  3. 「求職者」とは、求人情報の閲覧、保存、スワイプ、応募、企業との連絡その他求職活動を目的として本サービスを利用する個人をいいます。
  4. 「企業」とは、本サービス上に求人情報を掲載し、または求職者に対して採用活動を行う法人、個人事業主その他の事業者をいいます。
  5. 「会員」とは、当社所定の方法により本サービスの利用登録を完了した利用者をいいます。
  6. 「アカウント」とは、会員が本サービスを利用するために作成または付与される利用資格をいいます。
  7. 「プロフィール情報」とは、氏名、年齢、生年月日、住所、電話番号、メールアドレス、写真、自己紹介、職歴、学歴、資格、スキル、希望職種、希望勤務地、希望年収その他求職者が登録する情報をいいます。
  8. 「求人情報」とは、企業が本サービスに掲載する求人、募集条件、業務内容、給与、勤務地、勤務時間、福利厚生、応募条件、企業情報その他採用に関する情報をいいます。
  9. 「投稿データ」とは、利用者が本サービス上で入力、送信、掲載、保存または表示する文章、画像、動画、ファイル、プロフィール情報、求人情報その他一切の情報をいいます。
  10. 「マッチング」とは、求職者による求人への興味表示、応募、企業による承諾、スカウトその他当社が定める条件を満たした状態をいいます。
  11. 「知的財産権」とは、著作権、特許権、実用新案権、意匠権、商標権その他一切の知的財産権およびそれらの登録等を受ける権利をいいます。

第3条（本サービスの内容）

  1. 本サービスは、求職者が求人情報をスワイプ等の操作により閲覧し、興味を持った求人に対して応募、保存、興味表示その他当社所定のアクションを行うことができる求人検索・求人マッチングプラットフォームです。
  2. 企業は、本サービス上に求人情報を掲載し、求職者からの応募または興味表示を受け取り、必要に応じて求職者と連絡を行うことができます。
  3. 当社は、求人情報、求職者情報およびそれらを利用したマッチング機能を提供するものであり、当社が別途明示しない限り、職業紹介、労働者派遣、採用代行または雇用契約の締結代理を行うものではありません。
  4. 当社は、求職者に対して採用、内定、面接機会、雇用条件、勤務開始、雇用継続その他企業との契約成立を保証しません。
  5. 当社は、企業に対して応募、採用、求職者の能力、経歴、勤務可能性、定着、成果その他採用活動の結果を保証しません。
  6. 本サービスの具体的内容、機能、利用条件、表示方法、アルゴリズムおよび仕様は、当社の裁量により随時変更されることがあります。

第4条（会員登録）

  1. 本サービスの利用を希望する者は、本規約に同意し、当社所定の情報を当社所定の方法で提供することにより、会員登録を申請することができます。
  2. 登録希望者は、真実、正確かつ最新の情報を登録しなければなりません。
  3. 登録情報に変更が生じた場合、会員は速やかに当社所定の方法により登録情報を変更するものとします。
  4. 当社は、登録希望者が次の各号のいずれかに該当すると判断した場合、登録または再登録を拒否することができ、その理由を開示する義務を負いません。
    1. 当社に提供した情報に虚偽、誤りまたは記載漏れがある場合
    2. 第三者になりすまして登録を行った場合
    3. 同一人物または同一企業が当社の承諾なく複数のアカウントを作成した場合
    4. 過去に本規約違反等により利用停止、登録取消しその他の措置を受けた者である場合
    5. 反社会的勢力等に該当し、または関与している場合
    6. 法令、公序良俗または本規約に違反するおそれがある場合
    7. その他当社が登録を適当でないと判断した場合

第5条（アカウント管理）

  1. 会員は、自己の責任においてアカウント、ID、パスワードその他認証情報を適切に管理するものとします。
  2. 会員は、アカウントを第三者に貸与、譲渡、売買、共有または担保提供してはなりません。
  3. 会員のアカウントを利用して行われた行為は、当該会員本人による行為とみなします。
  4. アカウントの管理不十分、使用上の過誤、第三者の使用、不正アクセス等によって生じた損害について、当社は当社に故意または重過失がある場合を除き責任を負いません。
  5. 会員は、アカウントの不正使用またはそのおそれを認識した場合、直ちに当社へ通知するものとします。

第6条（求職者による利用）

  1. 求職者は、自己の責任において本サービスを利用し、求人情報の閲覧、応募、企業との連絡その他求職活動を行うものとします。
  2. 求職者は、プロフィール情報、職務経歴、保有資格、希望条件その他登録情報について、真実、正確かつ最新の内容を登録するものとします。
  3. 求職者は、虚偽の経歴、資格、実績、勤務可能条件その他採用判断に影響を与える情報を登録、送信または表示してはなりません。
  4. 求職者は、企業との面接、雇用契約、労働条件確認その他採用に関する判断を自己の責任で行うものとします。
  5. 求職者が企業との間で雇用契約、業務委託契約その他の契約を締結する場合、当該契約の当事者は求職者と企業であり、当社は当該契約の当事者となりません。

第7条（企業による利用）

  1. 企業は、自己の責任において求人情報を掲載し、採用活動を行うものとします。
  2. 企業は、求人情報について、真実、正確、最新かつ法令に適合した内容を掲載しなければなりません。
  3. 企業は、労働条件、業務内容、給与、勤務地、勤務時間、雇用形態、契約期間、試用期間、社会保険、休日、残業、受動喫煙防止措置その他求職者の判断に影響を与える事項を、誤解が生じないよう明確に表示するものとします。
  4. 企業は、求人情報に変更が生じた場合、速やかに求人情報を修正または掲載停止するものとします。
  5. 企業は、応募者または求職者から取得した個人情報を、採用選考その他あらかじめ明示した目的の範囲内で適切に取り扱うものとします。
  6. 企業は、当社が求人情報の内容確認、修正依頼、掲載停止、削除その他必要な措置を行うことに同意します。

第8条（求人情報の掲載および審査）

  1. 当社は、企業が掲載しようとする求人情報について、掲載前または掲載後に審査、確認、修正依頼、掲載停止または削除を行うことができます。
  2. 当社は、求人情報の掲載を承認した場合であっても、当該求人情報の適法性、正確性、完全性、最新性、安全性、採用条件その他一切を保証するものではありません。
  3. 当社は、求人情報が次の各号のいずれかに該当すると判断した場合、企業に通知することなく、当該求人情報を修正、非表示、掲載停止または削除することができます。
    1. 法令または公序良俗に違反する場合
    2. 虚偽、誇大、誤認を招く表示が含まれる場合
    3. 労働条件が明確でない場合
    4. 求職者に不利益または危険を及ぼすおそれがある場合
    5. 当社または第三者の信用を毀損するおそれがある場合
    6. 本規約または当社の掲載基準に違反する場合
    7. その他当社が不適切と判断する場合
  4. 前項の措置により企業または第三者に損害が生じた場合でも、当社は当社に故意または重過失がある場合を除き責任を負いません。

第9条（スワイプ、表示順およびレコメンド）

  1. 求職者は、本サービス上で求人情報をスワイプ、保存、非表示、応募、興味表示その他当社所定の方法により操作することができます。
  2. 求人情報の表示順、表示回数、推薦、非表示、検索結果その他の表示内容は、利用者の登録情報、利用履歴、スワイプ履歴、位置情報、希望条件、企業の設定、求人情報の内容、当社のアルゴリズムその他当社が定める基準により決定される場合があります。
  3. 当社は、求人情報の表示順、表示回数、推薦結果、マッチング結果その他アルゴリズムによる結果について、特定の成果を保証しません。
  4. 当社は、本サービスの品質向上、不正防止、利便性向上、求人表示の最適化等のため、利用者の行動履歴その他の情報を分析することがあります。

第10条（応募、マッチングおよび採用）

  1. 求職者による応募、興味表示、スワイプその他の行為は、採用、内定、面接確約または雇用契約の成立を意味するものではありません。
  2. 企業による返信、スカウト、マッチング承諾その他の行為は、採用、内定または雇用契約の成立を意味するものではありません。
  3. 採用選考、面接、労働条件の提示、内定、雇用契約の締結、勤務開始その他採用に関する一切の事項は、求職者と企業の責任において行われます。
  4. 求職者および企業は、採用条件、労働条件、契約内容その他重要事項について、自己の責任で確認するものとします。
  5. 当社は、求職者と企業との間で生じたトラブル、紛争、損害、労働問題、内定取消し、採用取消し、雇用条件の相違その他一切について、当社に故意または重過失がある場合を除き責任を負いません。

第11条（料金および支払）

  1. 本サービスの利用料金、支払方法、支払時期、契約期間、解約条件その他有料機能に関する事項は、当社が別途定め、本サービス上または申込画面に表示するものとします。
  2. 当社が無料で提供する機能について、当社はいつでも有料化、変更、制限または終了することができます。
  3. 企業が有料プランまたは有料オプションを利用する場合、企業は当社が定める料金を当社が指定する方法により支払うものとします。
  4. 支払済みの料金は、法令に別段の定めがある場合または当社が別途認めた場合を除き、返金されません。
  5. 利用者が支払を遅滞した場合、当社は本サービスの利用停止、求人掲載停止、アカウント停止その他必要な措置を講じることができます。

第12条（禁止事項）

利用者は、本サービスの利用にあたり、次の各号の行為を行ってはなりません。

  1. 法令、公序良俗または本規約に違反する行為
  2. 犯罪行為または犯罪行為に関連する行為
  3. 当社、他の利用者または第三者の権利、利益、名誉、信用、プライバシー、知的財産権を侵害する行為
  4. 虚偽の情報を登録、掲載、送信または表示する行為
  5. 第三者になりすます行為

  7. 本サービスを通じて取得した情報を、本サービスの目的外で利用する行為
  8. 求職者、企業または第三者に対する嫌がらせ、脅迫、差別、誹謗中傷、つきまといその他迷惑行為
  9. 営業、勧誘、宣伝、広告、宗教活動、政治活動、投資勧誘、マルチ商法、ネットワークビジネスその他本サービスの目的に反する行為

  14. 当社が定める方法以外で本サービスにアクセスする行為
  15. その他当社が不適切と判断する行為

第13条（求人掲載に関する禁止事項）

企業は、求人情報の掲載または採用活動にあたり、次の各号の求人または行為を行ってはなりません。

  1. 虚偽、架空、誇大または実態のない求人
  2. 労働条件、給与、勤務地、雇用形態、業務内容その他重要事項が事実と異なる求人
  3. 労働基準法、職業安定法、男女雇用機会均等法、最低賃金法、労働者派遣法その他労働関係法令に違反する求人
  4. 年齢、性別、国籍、信条、社会的身分、障害、疾病その他法令上不適切な差別的条件を含む求人
  5. 反社会的勢力、違法行為、犯罪行為、いわゆる闇バイト、特殊詐欺、名義貸し、口座売買、違法な運搬行為その他違法または危険な業務に関する求人

  8. 求職者に金銭、物品、登録料、研修費、教材費その他名目を問わず不当な負担を求める求人
  9. 採用とは無関係な個人情報、機微情報または過度な情報を取得しようとする行為
  10. 本サービス外での不適切な誘導、直接取引、外部サービス登録、個人連絡先の取得を目的とする求人
  11. その他当社が不適切と判断する求人または採用活動
