JobSwipe 求人掲載ガイドライン（企業向け）

最終改定日：令和8年7月1日

はじめに

本ガイドラインは、株式会社Level Frontier（以下「当社」といいます。）が提供する求人マッチングサービス「JobSwipe」（以下「本サービス」といいます。）に求人を掲載する企業、団体、個人事業主その他の事業者（以下「掲載企業」といいます。）が遵守すべき事項を定めるものです。

掲載企業は、本ガイドライン、JobSwipe利用規約、プライバシーポリシーその他当社が定めるルールを遵守し、法令に適合した求人掲載および採用活動を行うものとします。

第1条（掲載資格）

本サービスに求人を掲載できるのは、次の各号のいずれかに該当する事業者とします。

  1. 法人
  2. 個人事業主
  3. その他当社が掲載を認めた事業者

当社は、掲載企業に対し、必要に応じて以下の資料の提出を求めることがあります。

  • 登記事項証明書
  • 本人確認書類
  • 営業許可証
  • 職業紹介事業許可証
  • 労働者派遣事業許可証
  • その他当社が必要と判断する資料

第2条（求人情報の正確性）

掲載企業は、掲載する求人情報について、次の各号を保証するものとします。

  1. 真実かつ正確であること
  2. 最新の内容であること
  3. 法令に違反しないこと
  4. 求職者に誤認を与えないこと
  5. 実際の労働条件と一致していること

掲載企業は、求人内容に変更が生じた場合、速やかに求人情報を修正または掲載停止するものとします。

第3条（必須掲載事項）

掲載企業は、求人情報において、少なくとも以下の事項を明確に記載するものとします。

  • 会社名または事業者名
  • 勤務地
  • 雇用形態
  • 業務内容
  • 勤務時間
  • 休日・休暇
  • 給与・報酬
  • 各種手当
  • 試用期間の有無および条件
  • 福利厚生
  • 応募条件
  • 選考方法
  • その他求職者の判断に重要な事項

第4条（給与・報酬表示）

給与または報酬を表示する場合、掲載企業は次の各号を遵守するものとします。

  1. 月給、時給、日給、年俸、業務委託報酬等の区分を明確にすること
  2. 固定残業代を含む場合は、金額、対象時間数、超過分の支給有無を明記すること
  3. インセンティブ、歩合、成果報酬がある場合は、算定条件を明記すること
  4. 最低保証額がある場合は、その金額および条件を明記すること
  5. 試用期間中に条件が異なる場合は、その内容を明記すること

「高収入」「誰でも稼げる」「月収○○万円以上可能」等の表現を使用する場合は、根拠を明確にし、誤認を招かないようにしなければなりません。

第5条（禁止求人）

掲載企業は、次の各号に該当する求人を掲載してはなりません。

1. 違法または違法のおそれがある求人

  • 法令に違反する業務
  • 犯罪行為に関与する業務
  • 詐欺、名義貸し、口座売買、本人確認書類の貸与等を含む業務
  • マネー・ロンダリングに関与するおそれのある業務

2. 公序良俗に反する求人

  • 売春、買春、援助交際その他性行為を目的とする業務
  • 違法風俗、違法カジノその他違法営業に関する業務
  • わいせつ、暴力、差別その他公序良俗に反する業務

3. 闇バイトその他危険性の高い求人

  • 業務内容を明示しない高額報酬求人
  • 荷物の受取、運搬、口座開設、名義貸し等を目的とする求人
  • SNSや匿名通信アプリのみで連絡を行わせる求人
  • 身分証、銀行口座、携帯電話等を預けさせる求人
  • 「秘密厳守」「即日高収入」「誰でも簡単」等を過度に強調する求人

4. 反社会的勢力に関する求人

  • 暴力団、暴力団関係企業、総会屋、特殊知能暴力集団その他反社会的勢力に関する求人
  • 反社会的勢力への利益供与に関する求人

5. 勧誘目的の求人

  • マルチ商法、ネットワークビジネス、MLMに関する求人
  • 宗教活動または政治活動への勧誘を目的とする求人
  • 投資、FX、暗号資産、情報商材等への勧誘を主目的とする求人

6. その他不適切な求人

  • 空求人、おとり求人、虚偽求人
  • 他社求人の無断転載
  • 個人情報の収集のみを目的とする求人
  • 無許可で職業紹介または労働者派遣に該当する行為を行う求人
  • その他当社が不適切と判断する求人

第6条（差別的表現の禁止）

掲載企業は、法令上認められる場合を除き、次の事項を理由として応募者を制限し、または差別的な表現を用いてはなりません。

  • 年齢
  • 性別
  • 国籍
  • 人種
  • 信条
  • 宗教
  • 社会的身分
  • 障害の有無
  • 婚姻状況
  • 家族構成
  • その他法令上差別的取扱いが禁止される事項

第7条（写真・画像・動画の掲載）

掲載企業は、求人情報に掲載する写真、画像、動画その他の素材について、自己が必要な権利を有していることを保証するものとします。

次の各号に該当する素材の掲載は禁止します。

  1. 第三者の著作権、肖像権、商標権その他の権利を侵害するもの
  2. 実際の職場環境、業務内容、従業員の状況を著しく誤認させるもの
  3. AI生成画像等を用いて、実在の職場であるかのように誤認させるもの
  4. 公序良俗に反するもの
  5. その他当社が不適切と判断するもの

第8条（応募者情報の取扱い）

掲載企業は、本サービスを通じて取得した応募者情報を、採用活動に必要な範囲でのみ利用するものとします。

掲載企業は、応募者情報について、次の各号の行為を行ってはなりません。

  1. 第三者への提供、開示、販売
  2. 採用活動以外の営業、勧誘、広告配信への利用
  3. 本人の同意を得ない外部サービスへの登録
  4. 不必要な長期間の保存
  5. その他個人情報保護法その他の法令に違反する行為

第9条（企業アカウントの管理）

掲載企業は、自己の責任において企業アカウントを適切に管理するものとします。

掲載企業は、企業アカウントを第三者に譲渡、貸与、売買、共有してはなりません。ただし、当社が認める管理者権限の付与等はこの限りではありません。

企業アカウントを利用して行われた行為は、当該掲載企業による行為とみなします。

第10条（採用活動における遵守事項）

掲載企業は、求職者に対して誠実かつ適切に対応するものとします。

次の各号の行為は禁止します。

  1. 応募者への不当な放置
  2. 面接後の合理的理由のない無断連絡停止
  3. 求人情報と異なる条件の強要
  4. 威圧的、侮辱的、差別的な対応
  5. セクシュアルハラスメント、パワーハラスメントその他のハラスメント
  6. 応募者に対する不当な金銭請求
  7. その他求職者保護の観点から不適切な行為

第11条（外部誘導の制限）

掲載企業は、当社が認める場合を除き、本サービスの趣旨に反して、求職者を外部サイト、外部アプリ、SNS、匿名通信アプリ等へ不適切に誘導してはなりません。

特に、求人内容の詳細を本サービス上で明示せず、外部連絡先でのみ説明する行為は禁止します。

第12条（当社による審査）

当社は、求人情報の掲載前または掲載後に、求人内容、企業情報、応募者対応状況その他必要な事項を審査できるものとします。

当社は、審査の結果、必要と判断した場合、掲載企業に対して求人内容の修正、資料提出、説明その他必要な対応を求めることができます。

第13条（掲載停止・削除）

当社は、次の各号のいずれかに該当すると判断した場合、事前通知なく求人情報の非公開、掲載停止、削除、企業アカウントの停止その他必要な措置を行うことができます。

  1. 本ガイドラインに違反した場合
  2. JobSwipe利用規約に違反した場合
  3. 法令違反またはそのおそれがある場合
  4. 虚偽、誤認、不適切な求人である場合
  5. 求職者から苦情または通報があった場合
  6. 求職者保護の観点から必要がある場合
  7. その他当社が掲載継続を不適切と判断した場合

第14条（通報・調査協力）

当社は、求職者その他第三者から通報、苦情、問い合わせを受けた場合、掲載企業に対して事実確認、資料提出、説明その他必要な協力を求めることができます。

掲載企業が合理的な期間内に協力しない場合、当社は求人掲載の停止その他必要な措置を行うことができます。

第15条（採用成立・応募数等の非保証）

当社は、本サービスにおいて、次の各号を保証しません。

  1. 求人の閲覧数
  2. 応募数
  3. マッチング数
  4. 採用成立
  5. 応募者の能力、経歴、適性、勤務継続
  6. 求人掲載による売上、利益その他の成果

第16条（紛争対応）

掲載企業と求職者との間で、採用、労働条件、面接、入社、勤務、退職その他に関して紛争が生じた場合、掲載企業は自己の責任と費用においてこれを解決するものとします。

当社は、当社に故意または重過失がある場合を除き、掲載企業と求職者との間の紛争について責任を負いません。

第17条（損害賠償）

掲載企業が本ガイドライン、利用規約または法令に違反し、当社、求職者その他第三者に損害を与えた場合、掲載企業はその一切の損害を賠償するものとします。

第三者から当社に対して請求、苦情、行政指導、調査その他の対応が生じた場合、掲載企業は自己の責任と費用においてこれを解決し、当社に損害を与えないものとします。

第18条（ガイドラインの改定）

当社は、必要に応じて本ガイドラインを変更することができます。

変更後のガイドラインは、当社が本サービス上に掲載した時点または当社が別途定める時点から効力を生じるものとします。

変更後に掲載企業が本サービスの利用または求人掲載を継続した場合、当該掲載企業は変更後のガイドラインに同意したものとみなします。
